# Para inicializar o projeto

## Modo Desenvolvimento

```bash
# Executar o Projeto
npm run dev
```

## Modo Produção

```bash
# Construir o projeto
npm run build
```

```bash
# Executar o Projeto
npm run start
```

Abrir [http://localhost:3000](http://localhost:3000) com seu navegador.
